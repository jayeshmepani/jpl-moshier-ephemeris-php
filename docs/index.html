JME Native Wrapper

JPL Moshier Ephemeris PHP FFI

This package exposes the project-owned jme_* native API through PHP FFI. Swiss compatibility names are not the primary contract and should not be used as the package surface.

Current Direction

Use JmeEph\FFI\JmeEphFFI, jme_* methods, and JME_* constants. Native implementation status lives in the companion C repository.

Package Notes

This wrapper tracks the JME native library bundled with the package. Exact availability depends on the native implementation, not on legacy Swiss compatibility names.
